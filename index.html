```html id="przyklady123" Przykłady PHP – Tablice i Funkcje

📊 Przykłady działania funkcji

count()

$tab = ["Ala","Ola","Ela"];
echo count($tab);
Wynik: 3

sort()

$tab = [5,2,9,1];
sort($tab);
print_r($tab);
Wynik: [1,2,5,9]

rsort()

$tab = [5,2,9,1];
rsort($tab);
print_r($tab);
Wynik: [9,5,2,1]

asort()

$tab = ["a"=>5,"b"=>2,"c"=>9];
asort($tab);
print_r($tab);
Wynik: b=2, a=5, c=9

ksort()

$tab = ["c"=>5,"a"=>2,"b"=>9];
ksort($tab);
print_r($tab);
Wynik: a=2, b=9, c=5

file_get_contents()

$tekst = file_get_contents("plik.txt");
echo $tekst;
Wynik: zawartość pliku

file_put_contents()

file_put_contents("plik.txt", "Hello");
Wynik: zapisano "Hello" do pliku

file_exists()

if(file_exists("plik.txt")){
 echo "Istnieje";
}
Wynik: Istnieje

unlink()

unlink("plik.txt");
Wynik: plik usunięty

time()

echo time();
Wynik: np. 1712580000

date()

echo date("Y-m-d");
Wynik: 2026-04-08

👉 Demo (działa naprawdę)
```
